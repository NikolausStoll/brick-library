# Brick Library Add-on Assets

This directory serves as the home for Home Assistant metadata whenever the Brick Library personal inventory tracker ships as an add-on.

It mirrors the following structure:

- `config.yaml` – add-on metadata such as ports and supported architectures.
- `DOCS.md` – explains how to use the add-on metadata when packaging.
- `static/` – placeholder assets required by Home Assistant for branding and packaging.

Replace the contents of `static/` with real icons (e.g. `icon.png`) when the add-on is ready for packaging.
